CryptoSentry — Privacy Policy

Last updated: March 27, 2026

Summary

CryptoSentry does not collect, store, transmit, or sell any personal data. All your settings, watchlist, and alert configurations are stored locally on your device using Chrome's built-in storage API. Nothing leaves your browser.

Data Collection

CryptoSentry collects no personal information whatsoever. We do not collect names, email addresses, IP addresses, browsing history, wallet addresses, or any form of personally identifiable information.

Local Storage Only

All extension data — including your coin watchlist, alert settings, RSI thresholds, and preferences — is stored exclusively in your browser using the Chrome Storage API. This data never leaves your device and is not accessible to us or any third party.

Third-Party API Calls

CryptoSentry makes requests to the following public APIs to retrieve cryptocurrency market data:

CoinGecko API (api.coingecko.com) — for price, volume, and historical OHLC data
Alternative.me API — for the Fear & Greed Index

These API calls contain no personal information. They request only publicly available market data (e.g., the current price of Bitcoin). Please refer to each provider's own privacy policy for their data handling practices.

Wallet Access

CryptoSentry does not access, connect to, or interact with any cryptocurrency wallets. The extension has no capability to view, modify, or transact with your funds.

Permissions

CryptoSentry requests only the minimum Chrome permissions necessary to function:

storage — to save your watchlist and alert settings locally
alarms — to schedule periodic price checks
notifications — to deliver alert notifications
host permissions for API domains — to fetch market data from CoinGecko and Alternative.me

Analytics & Tracking

CryptoSentry does not include any analytics, telemetry, tracking pixels, or similar technologies. We have no way to know who installs or uses the extension.

Changes to This Policy

If this privacy policy is updated, the changes will be reflected on this page with an updated date. Significant changes will also be noted in the extension's changelog on the Chrome Web Store.
